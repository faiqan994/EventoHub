# EventoHub
EventoHub is a mobile application prototype designed to simplify event discovery and management. Users can browse events, view event details, book tickets, and manage their registrations. Organizers can create and manage events through a dedicated interface.

This repository currently contains the mobile application frontend/UI implementation and demonstration features developed as part of a university Mobile Application Development project. Some functionalities use sample or mock data and are intended for educational purposes.

Technologies

Kotlin
Android Studio
XML
Material Design Components
